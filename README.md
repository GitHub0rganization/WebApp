# linqs



## Setup

### Backend (database & api)
```go
// cd database/src/
$ make migrate/init
$ make migrate/up
$ docker-compose up
```
```go
// cd api/src/
$ make .env
$ make run
```

## Frontend (ui)
```bash
# cd ui

# install dependencies
$ yarn add

# start localhost
$ yarn start
```
